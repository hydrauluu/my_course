# Python: Глубокое погружение

**Продвинутый курс по Python для тех, кто хочет понять язык изнутри**

---

## 📋 О курсе

Этот курс — не просто очередное введение в Python. Мы разберём **внутреннее устройство языка**:
как работают дескрипторы, метаклассы, MRO, замыкания и другие механизмы,
которые обычно остаются за кадром.

### Для кого этот курс

- ✅ Вы уже знаете основы Python (переменные, циклы, функции, классы)
- ✅ Хотите понимать, **как это работает под капотом**
- ✅ Готовы разбираться с тонкостями протоколов и магическими методами
- ✅ Хотите писать более чистый и идиоматичный код

### Чему вы научитесь

- 🔹 Глубоко понимать объектную модель Python
- 🔹 Использовать дескрипторы для создания мощных API
- 🔹 Писать метаклассы и понимать, когда они действительно нужны
- 🔹 Работать с итераторами, генераторами и асинхронностью на продвинутом уровне
- 🔹 Понимать MRO и тонкости наследования
